Material de trabajo

En este curso aprenderas a usar las siguientes herramientas.

CDJ

CDJ es un término para describir un reproductor de CD fabricado por Pioneer Electronics, que funciona de manera similar a
un fonógrafo de vinilio para los propósitos de un DJ. Originalmente, el término se refería al CDJ-500 de Pioneer Electronics,
pero ahora abarca la línea completa de tocadiscos-CD de Pioneer.

Technics 1200

Bajo la marca Technics, la empresa producía una gran variedad de productos de alta fidelidad, como tocadiscos,
amplificadores, receptores, reproductores de cintas, reproductores de CD y altavoces, que se vendían en varios
países. Se concibió originalmente como una línea de equipos de audio de alta calidad.

Ableton Live

Ableton Live ACT está pensado tanto para la composición musical como para la música en directo. Su interfaz de
usuario consiste en una sola ventana con diferentes secciones. La sección principal se divide en dos tipos de vistas.

Traktor pro scratch

En 2003, Native Instruments expandió Traktor de software puramente. Se asociaron con Stanton Magnetics para desarrollar
Traktor Final Scratch : el software para el sistema de vinilo digital Final Scratch de Stanton (que usaba discos de vinilo
estampados con código de tiempo para controlar los MP3 ), mientras que Stanton desarrolló el hardware. [2] Esta asociación
también permitió a Native Instruments usar la funcionalidad de código de tiempo de Final Scratch en sus propios productos Traktor.
